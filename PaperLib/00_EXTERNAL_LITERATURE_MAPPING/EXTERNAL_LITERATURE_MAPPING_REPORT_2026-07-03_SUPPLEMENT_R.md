## 1. Hyper-Kamiokande: Construction On Track for 2028 First Data

### Source
Hyper-Kamiokande Collaboration status presentations (J-PARC PAC 39th, CERN seminars January–March 2026); Kralik 2026 (King's College London); Shiozawa 2025 (ICRR).

### Key Data

| Proton decay sensitivity (p→e⁺π⁰, 10 yr) | **> 1 × 10³⁵ years** |
| Mass hierarchy determination | 3σ within 2–5 years |

### What It Means for CQE
Hyper-Kamiokande is the **second time-critical neutrino experiment** (alongside DUNE 2029) that will test CQE neutrino predictions. Where DUNE probes the 1300 km baseline with LAr TPC technology, Hyper-K uses the 295 km baseline with water Cherenkov — providing **complementary systematic error profiles**. If CQE predicts a specific δ_CP value (e.g., from E8 discrete symmetry), both experiments must converge on that value or falsify the prediction.

The proton decay sensitivity is equally important. Grand unified theories (GUTs) predict proton lifetimes in the range 10³⁰–10³⁶ years. Super-Kamiokande has set limits of τ(p→e⁺π⁰) > 2.4×10³⁴ years. Hyper-K will probe beyond 10³⁵ years — the **only realistic experiment** to do so. If CQE contains GUT-like unification at the lattice-code-chain level, it must predict whether proton decay is observable or not.

## 2. Belle II / LHCb Flavor Physics: R(D*) Converges to SM; Bc+ LFU Anomaly Emerges

### Source
Belle II Collaboration (Moriond 2026); LHCb Collaboration (CERN seminar May 2026, Jiang et al.); CERN Courier May/June 2026.

### Key Data
| Observable | Result | SM Prediction | Significance |
|------------|--------|---------------|--------------|
| Belle II R(D*) | Consistent with SM | 0.299 ± 0.003 | 1.5σ |
| LHCb Bc+ → J/ψ ℓ⁺ν ℓ⁻ LFU | Deviation from SM | — | Significant tension |
| LHCb γ (CP angle) | First Run 3 measurement | Compatible with previous | — |
| Belle II projected ϕ₁ (β) precision | ∼0.2° at 50–150 ab⁻¹ | — | — |

### What It Means for CQE
The **B-physics landscape is shifting**. The long-standing R(D*)–R(D) anomaly (BaBar/Belle/LHCb vs. SM) was a key motivation for lepton-flavor-universality-violating (LFUV) new physics. Belle II's updated 2026 measurement shows R(D*) consistent with the SM at 1.5σ — **weakening the anomaly case**. However, the LHCb Bc+ LFU test (Jiang et al., May 2026) shows a **new deviation** in the heaviest B meson sector.

For CQE: If the framework predicted an R(D*) deviation to justify new physics, that prediction must be updated. If CQE predicts no LFUV (consistent with the SM), the Belle II result supports it — but the LHCb Bc+ deviation must then be addressed. CQE must state its prediction for **all flavor observables** (R(D*), R(K(*)), Bs→μ⁺μ⁻, B+→K⁺νν̄) or risk cherry-picking.

### CQE Integration Paragraph (Copy-Paste Ready)

> **CQE flavor predictions must address the evolving B-physics landscape**. Belle II's updated R(D*) measurement (Moriond 2026) is consistent with the Standard Model at 1.5σ [Belle II Collaboration, 2026], weakening the lepton-flavor-universality-violation case that motivated much BSM model-building. Predicting one observable while ignoring others is insufficient; the framework must predict the **entire flavor structure** or explicitly state its scope boundary.

## 4. SUSY Null Results at LHC: No Evidence Through Run 3

### Source
ATLAS SUSY searches (full Run 2 + early Run 3); PDG 2023 SUSY review; Ventura 2018 summary updated through 2026.

### Key Data
| SUSY Particle | Excluded Mass (95% CL) |
|---------------|------------------------|
| Gluino | > 2.0–2.45 TeV |
| First/second-gen squarks | > 1.0–1.9 TeV |
| Stop (third-gen squark) | > 0.6–1.2 TeV |
| Electroweak gauginos | > 0.4–1.1 TeV |
| Sleptons | > 0.7 TeV |

### What It Means for CQE
After **more than a decade of LHC data** (Run 1, Run 2, and early Run 3), **no evidence for supersymmetry has been found**. The excluded mass ranges now reach well above 1 TeV for colored sparticles and several hundred GeV for electroweak sparticles. The HL-LHC (3000 fb⁻¹) will extend these limits further, particularly for electroweakino and compressed-spectrum scenarios.

For CQE: If the framework predicts supersymmetric partners (e.g., from D4/E8 symmetry breaking), it must state their predicted masses. If the predicted masses lie below the LHC exclusion limits, the prediction is already falsified. If CQE predicts **no SUSY** (e.g., because the lattice code chain does not admit superpartners), this must be stated explicitly — the null results support the prediction, but only if the prediction was made before the exclusions.

### CQE Integration Paragraph (Copy-Paste Ready)

> **CQE must state its SUSY prediction explicitly in light of LHC null results**. The ATLAS and CMS collaborations have found no evidence for supersymmetry through the full Run 2 dataset and early Run 3 data [ATLAS SUSY summaries, 2023–2026; PDG SUSY review, 2023]. Excluded mass ranges now reach gluino >2 TeV, squarks >1 TeV, and electroweak gauginos >400–1100 GeV. CQE must either: (1) predict specific superpartner masses and show they are consistent with LHC exclusion limits (e.g., above the excluded ranges or in unprobed compressed spectra); (2) predict that no superpartners exist, citing the lattice-code-chain structure as the reason; or (3) explicitly exclude SUSY predictions from the framework's scope, stating that CQE does not address supersymmetry. Any claim of SUSY prediction must include the predicted mass spectrum and a comparison to HL-LHC projected sensitivities.

## 5. Microsoft Majorana 2: Topological Qubit Parity Lifetimes >20 Seconds

### Source
Microsoft Quantum team (June 2026); Roman Lutchyn seminar (Niels Bohr Colloquium, June 2026); The Quantum Insider (June 2026); arXiv:2502.12252 (Aasen et al. 2025).

### Key Data
| Parameter | Majorana 1 (Feb 2025) | Majorana 2 (June 2026) |
|-----------|----------------------|------------------------|
| Superconductor | Aluminum | **Lead** |
| Topological qubit count | 8 | Scaled (prototype) |
| Topological gap | Baseline | **>2× improvement** |
| Parity lifetime | Milliseconds | **>20 seconds** |
| Improvement factor | — | **>1000×** |
| Roadmap target | Scalable quantum computer | **2029** |

### What It Means for CQE
Microsoft's topological qubit program achieved a **>1000× improvement in parity lifetimes** by switching from aluminum to lead superconductors (Majorana 2, June 2026). This is still a **pre-commercial research program** — no fault-tolerant logical qubit has been demonstrated, and two-qubit gate fidelities remain undisclosed. However, the trajectory is rapid.

For CQE's NP-07 (Leech-lattice QEC): The QEC community now has **two competing paradigms** for fault tolerance:
1. **Active QEC** (Google Willow, QuEra, Quantinuum): High physical qubit overhead, real-time decoding, requires >50% encoding ratios
2. **Topological protection** (Microsoft Majorana): Inherent error protection via non-local encoding, potentially far lower overhead

CQE's Leech-lattice code is an **active QEC** approach. It must benchmark against both active-QEC competitors (QuEra >50% encoding) and topological alternatives (Microsoft's roadmap to 2029). If CQE claims the Leech-lattice code is optimal, it must compare overhead, latency, and logical error rates to both paradigms.

### CQE Integration Paragraph (Copy-Paste Ready)

> **CQE QEC claims must benchmark against both active and topological paradigms**. Microsoft's Majorana 2 processor (June 2026) achieved parity lifetimes >20 seconds — a >1000× improvement over Majorana 1 — by switching to lead superconductors [Microsoft Quantum, 2026; Lutchyn 2026]. While topological qubits remain pre-commercial, Microsoft's roadmap targets a scalable quantum computer by 2029 [arXiv:2502.12252]. CQE's NP-07 Leech-lattice QEC is an active error-correction approach that must be compared to **both** active-QEC benchmarks (QuEra >50% encoding, 10⁻¹³ logical error [Supp Q §3.4]) **and** topological-protection alternatives (inherent error suppression, potentially lower overhead). CQE must state: (1) the predicted physical-to-logical qubit overhead of the Leech-lattice code; (2) the decoder latency; (3) the logical error rate; and (4) whether topological protection could achieve the same logical performance with fewer physical qubits. If the Leech-lattice code is claimed to be optimal, the comparison must include all competing paradigms, not just active QEC.
