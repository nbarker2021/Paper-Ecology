# CQECMPLX Proper Papers — Rework Index

This directory contains a hand-crafted reworking of the CQECMPLX paper corpus into a coherent, layered identity. The goal is to pull together every paper that addresses the same underlying work and state the full scope and scale clearly, while keeping proven, open, and interpretive claims strictly separated.

## Reading model

- **Prestate / grounding papers** establish the contract before the numbered work begins.
- **Main numbered papers** carry distinct slices of the same structure.
- **Supplements** (non-integer named papers) expand, cross-walk, or modal-refine the numbered papers.
