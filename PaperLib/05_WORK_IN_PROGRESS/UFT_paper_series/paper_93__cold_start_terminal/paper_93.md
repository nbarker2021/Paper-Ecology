## 1.2. The Explicit Cold-Start Terminal Selection Algorithm (Theorem 1.2)

**Theorem 1.2 (Explicit cold-start terminal selection algorithm).** The cold-start terminal selection algorithm is specified as follows. Given an arbitrary state $S$ of size $n$ (a bitstring of length $n$), the algorithm computes the terminal form in $O(\log n)$ steps:

1. **Compute the initial fingerprint** $f_0(S)$: Apply the Rule 90 cold-start formula (Paper 2, Theorem 5.1) to compute the center bit at depth $n$:
   $$f_0(S) = \mathrm{lucas\_bit}(n, 0) = 1 \iff (n \text{ is even}) \land (k = n/2 \leq n) \land (k \,\&\, n) == k.$$
   This step takes $O(\log n)$ time via bitwise operations.

2. **Map to lattice depth**: Compute the lattice depth index $d = n \bmod 72$. Map $d$ to the nearest terminal depth in the lattice code chain:
   $$d_t = \arg\min_{d' \in \{1, 3, 7, 8, 24, 72\}} |d - d'|.$$
   This step takes $O(1)$ time (constant-time lookup in the 6-element chain).

3. **Select terminal form**: The terminal form is the Higgs state (weight $w=5$), independent of $d_t$. The Higgs state is the VOA state at weight 5, with mass $m_H = 5 \cdot \kappa \cdot \mathrm{SCALE} = 125.25$ GeV (Paper 16, Theorem 3.1).

4. **Compute terminal fingerprint**: The fingerprint of the terminal state is the McKay–Thompson coefficient $c_5$ (Paper 90, Theorem 2.1), which counts the degeneracy of the weight-5 state in the Monster VOA.

5. **Output**: Return the terminal form $(w=5, d_t)$ and the fingerprint $c_5$.

*Proof.* Step 1 is the Rule 90 cold-start formula, which is computable in $O(\log n)$ time (Paper 2, Theorem 5.2). Step 2 is a constant-time lookup in the lattice code chain (Paper 4, Theorem 5.1). Step 3 is the VOA weight assignment (Paper 16, Theorem 4.1). Step 4 is the McKay–Thompson coefficient (Paper 90, Theorem 2.1). The total complexity is $O(\log n) + O(1) + O(1) + O(1) = O(\log n)$. ∎

**Corollary 1.2.1 (The terminal depth is a lattice code chain depth).** The terminal depth $d_t$ is always one of the lattice code chain depths $\{1, 3, 7, 8, 24, 72\}$. The terminal form (weight 5) is independent of $d_t$; $d_t$ determines the "path" through the lattice, not the terminal form itself.

*Proof.* Direct from Theorem 1.2. The lattice code chain depths are the terminal depths (Paper 4, Theorem 5.1). The terminal form is the Higgs state (weight 5) regardless of the path. ∎

**Corollary 1.2.2 (The fingerprint map is a constant function).** The fingerprint map $F: \mathcal{S} \to \mathbb{N}$ from the state space $\mathcal{S}$ to the Monster coefficients is the constant function $F(S) = c_5$ for all $S$. This is because the terminal form is always the Higgs state, and the fingerprint of the Higgs state is $c_5$.

*Proof.* Direct from Theorem 1.2. The terminal form is always the Higgs state (weight 5), so the fingerprint is always $c_5$. ∎

*Proof.* Direct from Theorem 1.2. Step 1 (Rule 90 cold-start) takes $O(\log n)$ time (Paper 2, Theorem 5.2). Steps 2–4 (lattice depth lookup, terminal form selection, fingerprint computation) take $O(1)$ time each. The total is $O(\log n) + O(1) + O(1) + O(1) = O(\log n)$. ∎
