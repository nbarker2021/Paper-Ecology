### NEXP-2026-001 — Mechanical Memory Triggers Topological Ordering In Epithelia

**Publication:** Memory-driven topological ordering during the transition from
dormant to migrating epithelia.

**Source:** arXiv:2605.23339. Not found in local known-ID list.

**What I read:** The abstract frames a transition from quiescent epithelial
monolayers to coherent tissue-scale migration. The paper claims mechanical
memory in quiescent epithelia can trigger active stress release, causing rapid
topological ordering distinct from ordinary unjamming or flocking.

**New category:** active_matter_morphogenesis_memory.

**Why this is new to the chain:** Existing external chains discuss observer
state, exciton/particle-hole physics, QCD information, representation choice,
and lattice/VOA math. They do not appear to carry epithelial mechanical memory
as a primary external object.

**Tentative LCR placement pressure:**

| LCR component | External object to preserve |
|---|---|
| L | stored mechanical memory in the dormant tissue state |
| C | active stress-release event that drives migration |
| R | topological defects/order parameters that distinguish the transition from unjamming/flocking |
| T | migration/readout window over tissue-scale polarity |

**Possible open thing this may close:** A missing physical example where memory
is not symbolic storage but a latent mechanical state that becomes an active
transition driver.

**Derivation required before entry:** Extract the order parameters, defect
statistics, stress-memory variable, and transition criterion. Then test whether
LCR can represent "stored memory -> active release -> topological ordering"
without reducing it to ordinary agent flocking.

**Boundary:** Preprint/theoretical-experimental comparator. No promotion until
the transition variables and measurements are tabled.
