## Attachment Rule

Each external publication gets:

1. A subject category.
2. A nearest LCR topic or slot-family pressure.
3. Existing bound route, if one already exists in PEEP/CAM/Cursor outputs.
4. A derivation route if no bound LCR topic exists.
5. A non-promotion boundary. No citation alone closes an LCR claim.
